Крестики-нолики (Tic-Tac-Toe)

Простая консольная игра на Python для двух игроков. Игроки по очереди ставят X и O на поле 3×3, пока один не победит или не наступит ничья.

Возможности -
Игра для двух игроков в терминале  
Поле отображается с номерами свободных клеток (1–9)  
Проверка победы по строкам, столбцам и диагоналям  
Определение ничьей, если поле заполнено  

Запуск игры
1. Открываем терминал (ctrl+alt+t)
2. Вписываем эту строчку в терминал sudo apt install python3 (Вводим пароль, скачиваем python3)
3. Далее вводим в терминал git clone https://github.com/midd159/tic-tac-toe.git (Копируем данные с GitHub)
4. cd tic-tac-toe (Переходим в файл)
5. python3 end.py (Запускаем игру)
6. Наслаждаемся игрой

Правила игры

Игрок X ходит первым.
Введите число от 1 до 9, чтобы поставить символ в соответствующую клетку.  
Если клетка занята или ввод некорректный — программа попросит повторить ход.  
Побеждает игрок, собравший три символа подряд (по горизонтали, вертикали или диагонали).  
Если все клетки заняты и победителя нет — объявляется ничья.
